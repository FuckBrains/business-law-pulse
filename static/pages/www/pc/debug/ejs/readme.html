接口地址

<%= CONST.ssl ? 'https':'http' %>://<%= CONST.domain.api %>/[ api ]

[ api ] 为对应模块的接口



请求方式

POST （减少 XSS 攻击风险）



请求类型

Content-Type : application/json （普通）

Content-Type : multipart/form-data （表单）

User-Agent : [App]/[Version] ([Device]; [OS]; [Manufacturer]) [Client]/[Version] Market/[Market] NetType/[NetType] Language/[Locale]



返回类型

Content-Type : application/json

时间戳（13位整型）统一精确至毫秒



返回结构

{ "err" : 0, "msg" : "", "data" : {}, "log" : "" }


[ err ]
  0 => 正常
 -1 => 应用代码异常
 -2 => 未授权
 -3 => 版本不支持
N>0 => 业务逻辑异常


[ log ]
err = -1 时，输出异常日志


[ msg ]
服务器返回的消息文本（可空）


[ data ]
服务器返回数据的 JSON 结构体
